# Playwright TypeScript Framework

This workspace contains a starter automation framework built with Playwright and TypeScript.

## Structure
- src/base: shared base classes and test fixtures
- src/pages: page object classes
- src/tests: test specs
- src/utils: reusable helpers

## Install dependencies
If Node.js and npm are available, run:

```bash
npm install
npx playwright install
```

## Run tests
```bash
npm test
```

## Build
```bash
npm run build
```
